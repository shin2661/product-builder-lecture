
# Lotto Number Generator

## Overview

This is a simple, stylish web application to generate random lottery numbers. It provides a clean, modern interface for users to get their lucky numbers with a single click.

## Design & Features

### Visual Design
*   **Theme:** Modern, clean, and minimalist.
*   **Layout:** Centered, single-column layout for easy use on all devices.
*   **Colors:** A simple color palette with a primary accent color for the button and numbers.
*   **Typography:** Clear, readable sans-serif font.
*   **Iconography:** Uses a "dice" or "star" icon for the button to enhance visual appeal.
*   **Effects:** Subtle animations on button hover and when numbers are generated to provide a dynamic user experience.

### Features
*   **Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Display:** Displays the generated numbers in a clear, easy-to-read format.
*   **Interaction:** A single "Generate" button to trigger the number generation.
*   **Responsive:** The layout adapts to different screen sizes, making it mobile-friendly.
